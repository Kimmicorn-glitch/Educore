# **App Name**: EduCore

## Core Features:

- Multi-Subject Learning Tracks: Offers learning tracks for Python, English, Maths, and Physics.
- Linked Learning Flow: Connects each concept to a related coding activity to reinforce learning.
- Interactive Code Runner: Provides an embedded sandboxed Python environment using Pyodide for in-browser code execution.
- AI Chatbot Assistance: An integrated chatbot provides contextual lesson data to answer questions using its tool, or explain examples.
- Progress Dashboard: Tracks lesson completion, exercise attempts, and performance insights per subject.
- Language Translation: Converts lessons into multiple languages using the LibreTranslate API.
- Secure Authentication: Firebase Authentication for sign-in with Google, Email/Password, and Anonymous Demo options.
- Serverless Database: Uses Cloud Firestore to store user data, lessons, and progress in real time with role-based access control.
- AI-Powered Personalized Learning Tool: An AI tool analyzes a user's performance across different subjects and dynamically adjusts the difficulty of new lessons, targeting weak areas and reinforcing learned concepts. It monitors user interactions, response times, and error patterns to fine-tune the educational content.

## Style Guidelines:

- Primary color: Deep blue (#304FFE), representing knowledge, trust, and stability.
- Background color: Light blue (#E3F2FD), providing a calm and focused learning environment.
- Accent color: Bright orange (#FF9100), used for interactive elements and important calls to action to grab attention.
- Headline font: 'Space Grotesk' sans-serif font to convey the technical concepts.
- Body font: 'Inter' sans-serif font is paired for longer content for better readability.
- Use minimalist, modern icons for subjects and actions.
- Clean and structured layout to facilitate easy navigation.
- Subtle transitions and feedback animations for user interactions.
